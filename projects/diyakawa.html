Diyakawa 1.0

A autonomous underwater vehicle developed for the SAUVC 2024 competition, focusing on fundamental underwater navigation, control, and system integration. The system was designed using cost-effective components to validate core functionalities such as mobility, sensing, and basic mission execution.

Key Features

  • 6 DOF underwater mobility using brushed DC thrusters
  • PVC + steel hybrid structure with waterproofing
  • Raspberrypi and Pre-built image processing
  • Arduino-based control system with IMU and magnetometer

Achievements

  • Qualified among top 42 teams out of 103 via video submission
  • Selected into top 21 finalists
  • Achieved 11th place overall at SAUVC 2024

Diyakawa 2.0

An advanced and improved version of the Diyakawa platform, developed by addressing the limitations of Diyakawa 1.0. This system focuses on full autonomy, robustness, and high-performance underwater operation, integrating modern robotics frameworks and optimized mechanical design.

Key Features

  • Brushless thrusters
  • ROS2-based architecture with jetson nano

Achievements

  • Improved system performance
  • Achieved 10th place overall at SAUVC 2025
