Truck Alignment System

Project Overview

The Truck Alignment System is a computer vision-based solution developed to assist container terminal operators in accurately positioning trucks during loading and unloading operations. Improper truck alignment can increase crane cycle times, reduce operational efficiency, and create safety risks. The system automatically detects the truck position using a camera and provides real-time alignment guidance to the driver through an external display system.

Problem Statement

At container terminals, trucks must stop at precise locations to allow cranes to safely pick up or place containers. Manual alignment depends heavily on driver skill and often results in positioning errors, causing delays and additional crane movements. The objective of this project was to develop an automated system capable of continuously monitoring truck position and guiding drivers toward the correct alignment position.

System Architecture

  • Industrial camera captures real-time video of incoming trucks.
  • NVIDIA Jetson processes image data using computer vision algorithms.
  • YOLO object detection identifies and tracks the truck.
  • Distance and offset calculations determine alignment accuracy.
  • Arduino controls the external LED display system.
  • Ethernet communication enables fast data exchange between Jetson and Arduino.
  • Drivers receive alignment instructions through a visual display interface.

Key Features

  • Real-time truck detection and tracking.
  • Automatic lane identification.
  • Precise alignment guidance for drivers.
  • Low-latency image processing using Jetson.
  • Multi-threaded software architecture.
  • Reliable Ethernet communication.
  • User-friendly operator display system.
  • Scalable design for multiple lanes.

Technologies Used

Python YOLO OpenCV NVIDIA Jetson Arduino Uno Ethernet Communication Multithreading Computer Vision

Engineering Challenges

  • Maintaining accurate truck detection under varying lighting conditions.
  • Achieving real-time performance while processing video streams.
  • Ensuring reliable communication between Jetson and Arduino.
  • Handling different truck sizes and vehicle positions.
  • Reducing latency between detection and display updates.

Results & Impact

The developed system provides real-time alignment guidance, reducing driver positioning errors and improving operational efficiency during container handling operations. The solution demonstrates the practical application of computer vision and embedded systems in port automation and smart logistics.
